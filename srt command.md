# The str Command


Used to compactly display the internal structure of an R object

It is especially well suited to compactly display the contents of lists.

#### str ( function )

```r
> str(str)
function (object, …)

> str(lm)
function (formula, data, subset, weights, na.action, method = "qr", model = TRUE, x = FALSE, y = FALSE, 
    qr = TRUE, singular.ok = TRUE, contrasts = NULL, offset, ...)  

```
